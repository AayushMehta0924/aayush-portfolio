# aayush-portfolio

Personal portfolio for **Aayush Mehta** — Data + AI + Cloud Engineer.

Live at: _coming soon (Vercel)_

## Stack

Next.js 16 (App Router) · TypeScript · Tailwind v4 · Framer Motion · GSAP · Lenis · Tone.js · cmdk · MDX · Geist fonts.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Status

Phase 1 — skeleton (deployable, persistent shell wired up). Content surfaces, terminal hero, page transitions, and command palette land in subsequent phases.
